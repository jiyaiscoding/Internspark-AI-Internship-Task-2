# Deep Learning Sentiment Analysis using TensorFlow

## Overview

This project develops a deep learning model for sentiment analysis using the IMDb Movie Reviews Dataset.

Framework:

* TensorFlow/Keras

Model Components:

* Embedding Layer
* Bidirectional LSTM
* Dense Layers

---

## Dataset

Dataset Source:

IMDb Dataset provided by TensorFlow.

No manual dataset download is required.

The dataset is downloaded automatically during execution.

---

## Environment Setup

### Create Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate environment:

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```
